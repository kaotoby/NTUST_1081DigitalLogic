## How to compile

Note: Command examples below are in Ubuntu 18.04, but the source code should work on all kinds of system.

### Install build tools

```shell {class="line-numbers"}
sudo apt-get install build-essential
```

### Build

```shell {class="line-numbers"}
make
```

## How to use

```
Usage: lsim [-v] PROGRAM BLIF_FILE STIMULI_FILE RESPONSE_FILE

Options:
    -v    Showing all debug information.
```

### Simple usage

```shell {class="line-numbers"}
./lsim simple.blif simple.sti simple.res
```

You can also output topology sorted sequence and simulation states with `-v` option.
> -v    Showing all debug information.

```shell {class="line-numbers"}
./lsim -v simple.blif simple.sti simple.res
```

The above will output `simple.res`.

## Topology sort algorithm

### Pseudo code

```csharp {class="line-numbers"}
TopologySort(gates, gateCount, neighbors)
    for i = 0 to gateCount
        if visited[i] = false
            dfsStack.push((false, i))

        while dfsStack is not empty
            node = dfsStack.pop()
            if node is processed
                sortedGates.push_back(gates[node])
                continue

            visited[node] = true
            dfsStack.push((true, node.second))

            foreach neighbor in neighbors[node]
                if not visited[next]
                    dfsStack.push((false, next))

    return sortedGates;
```
